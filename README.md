# bricolage-mysql

MySQL-related job classes for Bricolage batch job framework.

## Home Page

https://github.com/bricolages/bricolage-mysql

## Usage

Add following line in your Gemfile:
```
gem 'bricolage-mysql', require: 'bricolage-mysql'
```
